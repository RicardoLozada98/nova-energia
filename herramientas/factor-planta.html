Notas técnicas

Fuentes consultadas: NREL PVWatts v8 con dataset NSRDB (USA · cobertura América) · PVGIS SARAH3 (Comisión Europea · satélite Meteosat 2005-2023) · NASA POWER + modelo simplificado de Liu-Jordan. El promedio entrega un valor robusto cuando las 3 fuentes coinciden (desviación < 5 %). Una desviación > 10 % entre fuentes indica revisar terreno (microclima, sombreado, datos satelitales pobres en la zona). NASA POWER usa un modelo simplificado de irradiancia en plano (factor heurístico para Ecuador), sirve como tercera referencia, no para diseño definitivo. Para licitaciones serias validar con PVsyst usando archivo Meteonorm.
