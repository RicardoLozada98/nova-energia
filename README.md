# NOVA ENERGIA — Portal de Herramientas

Portal interno de ingeniería para sistemas solares.

## Herramientas disponibles
- **Geoportales eléctricos**: acceso a EEQ, CNEL, Centrosur (Ecuador)
- **Caída de voltaje y pérdidas**: cálculo de ΔV con tablas PV WIRE, SUPERFLEX, RV-K
- **Factor de planta**: mapa interactivo + cálculo PVGIS + análisis económico

## En desarrollo
- Dimensionamiento PV (a partir de consumo)
- Análisis OCR de planilla eléctrica

## Acceso
Ingresa por `login.html`. Usuario master configurado en `assets/js/auth.js`.

## Stack
HTML + CSS + JavaScript puros. Sin backend. Librerías vía CDN: Leaflet, Leaflet-Draw, Turf.js.
